# LangGraph Portfolio

A collection of agentic AI projects built with [LangGraph](https://github.com/langchain-ai/langgraph), demonstrating state-machine orchestration, multi-step reasoning, tool use, and self-correcting pipelines.

## Projects

| # | Project | Description | Key Concepts |
|---|---------|-------------|--------------|
| 01 | [Adaptive RAG](projects/01_adaptive_rag/) | Intelligent query routing across vector store, web search, and direct generation with self-correction | Conditional routing, RAG, document grading, hallucination detection |

## Getting Started

Each project is self-contained with its own `requirements.txt` and `README.md`. Navigate to a project directory and follow the setup instructions there.

### Prerequisites

- Python 3.10+
- An [OpenAI API key](https://platform.openai.com/api-keys)
- A [Tavily API key](https://tavily.com/) (for projects using web search)
